How to Build a Minimal Portfolio

Step 1: Set Up Your Project

Create a folder for your portfolio project. Inside it, create these files:

  • index.html → your main HTML file
  • style.css → your CSS file

Your folder structure should look like this:

portfolio/
│
├── index.html
└── style.css

Step 2: Write the HTML

Here’s a simple example of index.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Minimal Portfolio</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <h1>My Name</h1>
        <p>Web Developer | Designer | Programmer</p>
    </header>

    <section id="about">
        <h2>About Me</h2>
        <p>Hi! I'm a web developer passionate about creating clean and functional websites.</p>
    </section>

    <section id="projects">
        <h2>Projects</h2>
        <div class="project">
            <h3>Project One</h3>
            <p>A brief description of the project.</p>
        </div>
        <div class="project">
            <h3>Project Two</h3>
            <p>A brief description of the project.</p>
        </div>
    </section>

    <section id="contact">
        <h2>Contact</h2>
        <p>Email: <a href="mailto:you@example.com">you@example.com</a></p>
    </section>

    <footer>
        <p>&copy; 2025 My Name</p>
    </footer>
</body>
</html>

Step 3: Add Minimal CSS

Here’s a clean, minimal style for style.css:

/* Reset default browser styles */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: 'Arial', sans-serif;
    line-height: 1.6;
    color: #333;
    background-color: #fefefe;
    padding: 20px;
}

header, section, footer {
    max-width: 800px;
    margin: 0 auto 40px;
    text-align: center;
}

h1 {
    font-size: 2.5rem;
    margin-bottom: 10px;
}

h2 {
    font-size: 1.8rem;
    margin-bottom: 15px;
}

.project {
    margin-bottom: 20px;
}

a {
    color: #007acc;
    text-decoration: none;
}

a:hover {
    text-decoration: underline;
}

footer {
    font-size: 0.9rem;
    color: #777;
}

Step 4: Open in Browser

  • Save your files.
  • Open index.html in your browser to see your minimal portfolio.
